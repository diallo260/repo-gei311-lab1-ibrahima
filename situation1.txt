 
Étapes suivies pour résoudre la Situation 1 :

1. Membre A crée un dépôt Git local avec la même structure que la partie 1.
   Commande : git init

2. Membre A décide de collaborer avec B et ils choisissent GitHub pour la gestion des versions.

3. Membre A crée un nouveau repository vide sur GitHub (sans README ni .gitignore).

4. Membre A associe son dépôt local avec le dépôt distant GitHub.
   Commande : git remote add origin https://github.com/utilisateur/nom-du-repo.git

5. Membre A pousse son projet local sur GitHub.
   Commande : git push -u origin main

6. Membre B peut maintenant cloner le projet depuis GitHub avec :
   Commande : git clone https://github.com/utilisateur/nom-du-repo.git
